## Prerequisites

- Git
- NodeJs
- npm

## Getting Started

- Install dependencies

```sh
npm install
```

### Run Tests

```sh
npm run test
```

### Generate Coverage Report

```sh
npm run coverage
```

### Deploy to Localhost

```sh
npm run deploy
```

### Deploy to Mainnet and Testnet

```sh
# Sepolia Testnet
npm run deploy:sepolia
```


```sh
# Ethereum Mainnet
npm run deploy:mainnet
```
